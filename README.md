# Full-Stack Weathermood Application

## Installation

client 

1. Clone the repo
   ```sh
   git clone  
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the app
    ```sh
    npm start
    ```
4. Open the app in your browser
    ```sh
    http://localhost:3030/
    ```

server

1. run schem.js 
   ```sh
    node src/model/schem.js
    ```
2. run server.js
    ```sh
    node src/server.js
    ```
